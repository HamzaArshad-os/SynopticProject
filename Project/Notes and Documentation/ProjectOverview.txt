A REST-API automated test generation tool. That takes in a YAML file openapi specification
and will return a set of tests in a supported language.

Supported languages:
Javascript

Supported Testing frameworks:
Chai
MockData is generated in:
JSON
MockData is generated using JSON-Schema-Faker.


The aim of this tool is to behave in a plug and play manner. Where a
user is able to give a specification and be returned a  set of tests. The user should be able to just copy paste these tests into there test directory and with minimal modifcations be able to use them in htere preexisiting testing enviornment.

This tool will geenrate valid and invalid mockData for testing purposes.
